Speech-to-Text Summarization

Allows users to record a live voice, convert it to text using Google Speech Recognition, and then perform TextRank-based summarization to summarize the voice content. Built using Python, NLTK, and Tkinter as the GUI, the app also presents a word count comparison between the original text and the summarized result in a visual form (bar graph). Key features include speech-to-text, text cleaning, tokenization, word frequency counting, and visualization of the summary results.

Project

CT Scan Edge Detection

Applied spatial filtering to CT scan images to clarify structures and detect edges. The image is processed with Gaussian Blur to reduce noise, followed by Sobel operator to extract horizontal and vertical edges. The results are displayed as a three-panel visualization: the original image, the Gaussian filtering result, and the edge detection result using Sobel. This project is useful for understanding the initial stages of preprocessing in medical image analysis.

Project

Image Alignment with ORB

Image matching and alignment using ORB (Oriented FAST and Rotated BRIEF) to detect features between two object images. Images are matched with Brute-Force Matcher using Hamming distance, then Homography transformation is calculated using RANSAC method. Display the 50 best feature matches, calculate the inliers and outliers, generate an overlay image from the alignment results. This process is useful in applications such as image stitching and object tracking.
